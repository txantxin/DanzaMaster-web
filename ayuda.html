Documentación inicial

Ayuda de Danza Master

Esta página sirve como base para ir creando un manual sencillo del programa. Los textos son provisionales y se podrán ampliar con instrucciones reales, capturas y ejemplos.

Primeros pasos

1. Configurar la academia

Revisar los datos principales de la academia, carpetas de trabajo, impresoras, documentos, copias de seguridad y preferencias generales.

2. Dar de alta alumnos y profesores

Crear las fichas principales con los datos necesarios para la gestión diaria: alumnos, tutores, autorizados, docentes y disciplinas.

3. Crear clases y horarios

Preparar las clases, asignar profesor, aula, horario, precios y condiciones necesarias antes de activarlas.

4. Activar clases

Una vez preparada una clase, se activa para el curso correspondiente y se inscriben los alumnos que correspondan.

Procesos frecuentes

Alta de alumno

Proceso para registrar un nuevo alumno, sus datos familiares y documentos asociados.

Cobros y recibos

Gestión de conceptos pendientes, cobros realizados y generación de recibos.

Facturas

Creación de facturas a partir de recibos y datos fiscales del destinatario.

Copias de seguridad

Protección de datos mediante copias manuales y automáticas.

Pendiente de completar
