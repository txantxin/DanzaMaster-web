Qué es Danza Master

Danza Master es un software de gestión creado para facilitar el trabajo diario de una academia de danza. La web sirve como punto de presentación, documentación básica y contenedor de ayuda para explicar el funcionamiento general del programa.

Primera versión de la web: esta página es una base inicial. Más adelante se pueden añadir capturas reales, manuales, vídeos, descargas, preguntas frecuentes o enlaces privados.

Módulos principales

Alumnos

Fichas de alumnos, datos familiares, tutores, autorizados, documentos y fotografías.

Clases

Organización de clases, packs, horarios, aulas, profesores y grupos activos.

Economía

Conceptos pendientes, cobros, recibos, facturas, descuentos y documentos emitidos.

Profesorado

Gestión de docentes, disciplinas, condiciones profesionales y panel de profesorado.

Asistencia

Control de fichajes, incidencias, terminales y revisión de registros de asistencia.

Copias de seguridad

Sistema de copias manuales y automáticas para proteger los datos de la academia.
